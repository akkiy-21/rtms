# Skill: 日次設備解析レポート PDF 生成

## 概要

`chart_skill.md` で生成したチャートPNGと解析結果テキストを、A4 PDFレポートに組み立てる。  
依存ライブラリは `matplotlib`（`chart_skill.md` と共通）のみ。追加インストール不要。

---

## ページ構成

| ページ | 向き | 内容 |
|---|---|---|
| 1 | 縦 | 表紙・KPI表 |
| 2〜N | 横 | チャート（1枚/ページ）。存在しないファイルはスキップ |
| 最終 | 縦 | 所見・考えられる要因・追加確認事項・参照データ |

---

## 入力仕様

### `report_data` dict

| キー | 型 | 説明 |
|---|---|---|
| `equipment_name` | str | 設備名 |
| `target_date` | str | 対象日 "YYYY-MM-DD" |
| `kpi` | dict[str, str] | KPI辞書。例: `{"可動率": "85.2%", "OEE": "72.1%"}` |
| `findings` | list[str] | 所見（箇条書き文字列リスト） |
| `factors` | list[str] | 考えられる要因（箇条書き） |
| `action_items` | list[str] | 追加確認事項（箇条書き） |
| `data_note` | str | 参照データ・備考 |

### `chart_paths` list

`chart_skill.md` が出力するPNGファイルのパスを順番通りに渡す。

```python
chart_paths = [
    "production_alarm_trend.png",
    "operation_composition.png",
    "alarm_pareto.png",
    "oee_availability_dashboard.png",
    "cycle_time_trend.png",      # 存在しない場合は自動スキップ
    "operation_timeline.png",
]
```

---

## レイアウト仕様

### 全ページ共通

| 要素 | 仕様 |
|---|---|
| ヘッダー | 濃紺バー (#1a3a5c)。左:チャート名/タイトル（白bold）、右:日付（白）|
| フッター | 薄灰バー (#efefef)。左:Generated by RTMS（灰）、右:ページ番号（灰）|
| ヘッダー高さ | 縦向き 6%、横向き 8% |
| フッター高さ | 4% |

### Page 1（表紙 + KPI表）

| 要素 | 仕様 |
|---|---|
| figsize | A4縦 (8.27, 11.69) |
| タイトル | "日次設備解析レポート"（fontsize=22, bold, #1a3a5c） |
| サブタイトル | 設備名（fontsize=15）、日付（fontsize=12）|
| 区切り線 | タイトル下に薄灰 FancyArrowPatch |
| KPI表 | ヘッダー残余スペースを動的に計算して配置。ヘッダー背景 #4c78a8（白文字）、偶数行 #f0f4f8 |
| セル高さ | `1.0 / (KPI行数 + 1)` で均等割り（行数に応じて自動調整） |

### チャートページ（横向き）

| 要素 | 仕様 |
|---|---|
| figsize | A4横 (11.69, 8.27) |
| ヘッダー左文字 | `CHART_TITLES` でファイル名→日本語チャート名に変換して表示 |
| チャート表示 | `fig.add_axes([0.01, bot+0.01, 0.98, top-bot-0.02])`（ヘッダー・フッターを除いた全域） |
| dpi | 150 |

### テキストページ（縦向き）

| 要素 | 仕様 |
|---|---|
| figsize | A4縦 (8.27, 11.69) |
| コンテンツ軸 | `[0.05, bot+0.01, 0.90, top-bot-0.02]`（xlim=0-1, ylim=0-1） |
| セクションヘッダー | `FancyBboxPatch` で塗り背景バー＋左端に「■ タイトル」 |
| セクション配色 | 所見: #4c78a8 / #e8f0f8 / 要因: #d62728 / #fdecea / 確認事項: #f4a700 / #fff8e6 / 備考: #555555 / #f5f5f5 |
| 箇条書き | `textwrap.fill(width=28)` で折り返し（全角日本語に対応）、先頭「• 」付き |
| ページ溢れ | y 座標 0.04 未満で自動改ページ |

> **重要：**
> - ヘッダー・フッターは `fig.add_axes()` で専用軸として描画する（patches の zorder 問題を回避）
> - KPI表の行高さは `row_h = 1.0 / (len(rows) + 1)` で均等割りし、各セルに `set_height(row_h)` を適用する
> - チャートページのチャート名は `CHART_TITLES` dict で管理し、未登録ファイルはステム名をそのまま使用する

## Pythonスクリプト

```python
import textwrap
import matplotlib.pyplot as plt
import matplotlib.patches as mpatches
from matplotlib.backends.backend_pdf import PdfPages
from pathlib import Path

# ── 共通設定（chart_skill.md の共通設定ブロックと合わせて実行する）──
# フォント設定は chart_skill.md の「共通設定」セクション参照

A4_PORTRAIT  = (8.27, 11.69)
A4_LANDSCAPE = (11.69, 8.27)

OUTPUT_PDF = Path("daily_report.pdf")

# ── カラーパレット ──────────────────────────────────────────────────
COL_HDR_BG   = "#1a3a5c"   # ヘッダー背景（濃紺）
COL_HDR_TEXT = "#ffffff"   # ヘッダー文字（白）
COL_FTR_BG   = "#efefef"   # フッター背景（薄灰）
COL_FTR_TEXT = "#999999"   # フッター文字（灰）
COL_KPI_HDR  = "#4c78a8"   # KPI表ヘッダー
COL_KPI_ODD  = "#ffffff"
COL_KPI_EVEN = "#f0f4f8"

# セクション配色 {名称: (文字色, 背景色)}
SECTION_STYLE = {
    "所見":          ("#4c78a8", "#e8f0f8"),
    "考えられる要因":   ("#d62728", "#fdecea"),
    "追加確認事項":    ("#f4a700", "#fff8e6"),
    "参照データ・備考": ("#555555", "#f5f5f5"),
}

# ファイル名→チャートタイトル対応
CHART_TITLES = {
    "production_alarm_trend":    "Chart 1: 30分別 生産推移とアラーム件数",
    "operation_composition":     "Chart 2: 操業・ロス時間構成",
    "alarm_pareto":              "Chart 3: アラーム Pareto",
    "oee_availability_dashboard": "Chart 4: OEE・可動率 ダッシュボード",
    "cycle_time_trend":          "Chart 5: サイクルタイム推移",
    "operation_timeline":        "Chart 6: 稼働タイムライン",
}


def _make_header_footer(fig, eq, date, page_num,
                        chart_title=None, is_landscape=False):
    """ヘッダー軸（濃紺バー）とフッター軸（薄灰バー）を描画する。
    Returns: (content_top, content_bottom) — コンテンツ配置可能領域の上端・下端（figure座標）
    """
    hdr_h = 0.08 if is_landscape else 0.06
    ftr_h = 0.04

    # ── ヘッダー ──
    ax_h = fig.add_axes([0, 1 - hdr_h, 1, hdr_h])
    ax_h.set_facecolor(COL_HDR_BG)
    ax_h.axis("off")
    left_text = chart_title if chart_title else f"{eq}　日次設備解析レポート"
    ax_h.text(0.02, 0.5, left_text,
              transform=ax_h.transAxes,
              color=COL_HDR_TEXT, fontsize=10, va="center", fontweight="bold")
    ax_h.text(0.98, 0.5, date,
              transform=ax_h.transAxes,
              color=COL_HDR_TEXT, fontsize=10, va="center", ha="right")

    # ── フッター ──
    ax_f = fig.add_axes([0, 0, 1, ftr_h])
    ax_f.set_facecolor(COL_FTR_BG)
    ax_f.axis("off")
    ax_f.text(0.02, 0.5, "Generated by RTMS Daily Analysis Skill",
              transform=ax_f.transAxes,
              color=COL_FTR_TEXT, fontsize=7, va="center")
    ax_f.text(0.98, 0.5, f"─  {page_num}  ─",
              transform=ax_f.transAxes,
              color=COL_FTR_TEXT, fontsize=8, va="center", ha="right")

    return 1 - hdr_h, ftr_h   # content_top, content_bottom


def _render_sections(pdf, sections, eq, date, start_page):
    """所見・要因・確認事項をテキストページに描画する。ページ溢れ時は自動改ページ。"""
    LINE_H   = 0.042   # 1行の高さ（コンテンツ軸座標）
    SEC_H    = 0.060   # セクションヘッダーバーの高さ
    GAP_SEC  = 0.030   # セクション間スペース
    GAP_ITEM = 0.010   # 項目間スペース
    MARGIN   = 0.05    # 左右マージン

    page_num = start_page

    def new_page():
        nonlocal page_num
        f = plt.figure(figsize=A4_PORTRAIT)
        f.patch.set_facecolor("white")
        top, bot = _make_header_footer(f, eq, date, page_num)
        # コンテンツ軸（ヘッダー・フッター間）
        a = f.add_axes([MARGIN, bot + 0.01, 1 - MARGIN * 2, top - bot - 0.02])
        a.set_xlim(0, 1)
        a.set_ylim(0, 1)
        a.axis("off")
        page_num += 1
        return f, a, 0.975

    fig, ax, y = new_page()

    for section_title, items in sections:
        if not items:
            continue

        col_text, col_bg = SECTION_STYLE.get(section_title, ("#333333", "#f5f5f5"))

        # ページ残量が足りなければ改ページ
        needed = SEC_H + LINE_H * max(1, len(items)) + GAP_SEC
        if y - needed < 0.03:
            pdf.savefig(fig, bbox_inches="tight")
            plt.close()
            fig, ax, y = new_page()

        # ── セクションヘッダーバー ──
        ax.add_patch(mpatches.FancyBboxPatch(
            (0, y - SEC_H), 1.0, SEC_H,
            boxstyle="round,pad=0.005",
            facecolor=col_bg, edgecolor=col_text,
            linewidth=1.2, clip_on=True, zorder=2,
            transform=ax.transData,
        ))
        ax.text(0.015, y - SEC_H / 2, f"■ {section_title}",
                color=col_text, fontsize=11, fontweight="bold",
                va="center", zorder=3)
        y -= SEC_H + 0.015

        # ── 箇条書き ──
        for item in items:
            wrapped = textwrap.fill(str(item), width=28).split("\n")
            for i, line in enumerate(wrapped):
                if y < 0.04:
                    pdf.savefig(fig, bbox_inches="tight")
                    plt.close()
                    fig, ax, y = new_page()
                prefix = "• " if i == 0 else "    "
                ax.text(0.025, y, prefix + line,
                        color="#333333", fontsize=9.5, va="top", zorder=2)
                y -= LINE_H
            y -= GAP_ITEM

        y -= GAP_SEC

    pdf.savefig(fig, bbox_inches="tight")
    plt.close()


def build_pdf_report(report_data, chart_paths, output_path=None):
    """
    Parameters
    ----------
    report_data : dict
        {
            "equipment_name": str,
            "target_date":    str,        # "YYYY-MM-DD"
            "kpi":            dict,       # {"可動率": "85.2%", ...}（値は文字列）
            "findings":       list[str],
            "factors":        list[str],
            "action_items":   list[str],
            "data_note":      str,
        }
    chart_paths : list[str | Path]
        生成済みチャートPNGパスリスト。存在しないファイルは自動スキップ。
    output_path : str | Path | None
        出力PDFパス。None の場合は OUTPUT_PDF を使用。
    """
    out  = Path(output_path) if output_path else OUTPUT_PDF
    eq   = report_data.get("equipment_name", "設備名")
    date = report_data.get("target_date", "")
    kpi  = report_data.get("kpi", {})
    page = 1

    with PdfPages(str(out)) as pdf:

        # ════════════════════════════════════════════
        # Page 1: 表紙 + KPI表
        # ════════════════════════════════════════════
        fig = plt.figure(figsize=A4_PORTRAIT)
        fig.patch.set_facecolor("white")
        top, bot = _make_header_footer(fig, eq, date, page)
        page += 1

        # タイトルブロック
        title_y = top - 0.02
        fig.text(0.5, title_y, "日次設備解析レポート",
                 ha="center", va="top", fontsize=22, fontweight="bold",
                 color="#1a3a5c", transform=fig.transFigure)
        fig.text(0.5, title_y - 0.09, eq,
                 ha="center", va="top", fontsize=15, color="#333333",
                 transform=fig.transFigure)
        fig.text(0.5, title_y - 0.15, f"{date}  JST",
                 ha="center", va="top", fontsize=12, color="#666666",
                 transform=fig.transFigure)

        # 区切り線
        fig.add_artist(mpatches.FancyArrowPatch(
            (0.10, title_y - 0.20), (0.90, title_y - 0.20),
            transform=fig.transFigure,
            arrowstyle="-", color="#aaaaaa", linewidth=1.2,
        ))

        # KPI表（タイトル下〜フッター上の残余スペースに動的配置）
        if kpi:
            tbl_top    = title_y - 0.23
            tbl_height = tbl_top - bot - 0.02
            ax_kpi = fig.add_axes([0.08, bot + 0.02, 0.84, tbl_height])
            ax_kpi.axis("off")
            rows  = [[k, str(v)] for k, v in kpi.items()]
            n_all = len(rows) + 1       # ヘッダー込み総行数
            row_h = 1.0 / n_all

            tbl = ax_kpi.table(
                cellText=rows,
                colLabels=["指標", "結果"],
                cellLoc="center",
                bbox=[0, 0, 1, 1],
            )
            tbl.auto_set_font_size(False)
            tbl.set_fontsize(10.5)
            tbl.auto_set_column_width([0, 1])

            for col in range(2):
                tbl[0, col].set_facecolor(COL_KPI_HDR)
                tbl[0, col].get_text().set_color("white")
                tbl[0, col].get_text().set_fontweight("bold")
                tbl[0, col].set_height(row_h)
            for row_idx in range(1, len(rows) + 1):
                bg = COL_KPI_EVEN if row_idx % 2 == 0 else COL_KPI_ODD
                for col in range(2):
                    tbl[row_idx, col].set_facecolor(bg)
                    tbl[row_idx, col].set_height(row_h)

        pdf.savefig(fig, bbox_inches="tight")
        plt.close()

        # ════════════════════════════════════════════
        # Pages 2-N: チャートページ（横向き）
        # ════════════════════════════════════════════
        for path in chart_paths:
            p = Path(path)
            if not p.exists():
                print(f"skip (not found): {p}")
                continue

            chart_title = CHART_TITLES.get(p.stem, p.stem)
            img = plt.imread(str(p))

            fig = plt.figure(figsize=A4_LANDSCAPE)
            fig.patch.set_facecolor("white")
            top, bot = _make_header_footer(fig, eq, date, page,
                                           chart_title=chart_title,
                                           is_landscape=True)
            page += 1

            ax_img = fig.add_axes([0.01, bot + 0.01,
                                   0.98, top - bot - 0.02])
            ax_img.imshow(img, aspect="auto")
            ax_img.axis("off")

            pdf.savefig(fig, bbox_inches="tight", dpi=150)
            plt.close()

        # ════════════════════════════════════════════
        # 最終ページ: テキスト（所見・要因・確認事項・備考）
        # ════════════════════════════════════════════
        sections = [
            ("所見",          report_data.get("findings",     [])),
            ("考えられる要因",   report_data.get("factors",      [])),
            ("追加確認事項",    report_data.get("action_items", [])),
            ("参照データ・備考", [report_data["data_note"]]
                               if report_data.get("data_note") else []),
        ]
        _render_sections(pdf, sections, eq, date, start_page=page)

    print(f"saved: {out}")


if __name__ == "__main__":
    report_data = {
        "equipment_name": "設備名",
        "target_date":    "YYYY-MM-DD",
        "kpi": {
            "総生産数":          "---",
            "良品数":            "---",
            "不良数":            "---",
            "良品歩留（品質率）":  "---",
            "計画稼働時間":       "--- 分",
            "正味稼働時間":       "--- 分",
            "停止ロス":          "--- 分",
            "性能ロス":          "--- 分",
            "可動率":            "---",
            "性能率":            "---",
            "OEE":              "---",
            "アラーム件数":       "---",
        },
        "findings":     ["所見1", "所見2", "所見3"],
        "factors":      ["要因候補1", "要因候補2"],
        "action_items": ["確認事項1", "確認事項2", "確認事項3"],
        "data_note":    "データソース：RTMS / テーブル：xxx / 備考：なし",
    }
    chart_paths = [
        "production_alarm_trend.png",
        "operation_composition.png",
        "alarm_pareto.png",
        "oee_availability_dashboard.png",
        "cycle_time_trend.png",
        "operation_timeline.png",
    ]
    build_pdf_report(report_data, chart_paths)
```

## 注意事項

- 日本語フォント設定は `chart_skill.md` の「共通設定」ブロックと同じセッションで実行すること
- `plt.imread()` はPNG・JPEGに対応。WebPは `Pillow` が別途必要
- KPI辞書の値は **文字列** で渡すこと（数値は `f"{x:.1f}%"` 等で変換してから渡す）
- チャートが多い場合は `PdfPages` のメタデータ（`pdf.infodict()`）にタイトル等を設定可能
